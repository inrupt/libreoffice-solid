Solid Pod Content Provider for LibreOffice

This extension enables LibreOffice to connect to Solid pods, allowing you to:
- Open documents directly from Solid pods
- Save documents to Solid pods  
- Browse Solid pod contents
- Authenticate using OAuth 2.0 + PKCE + DPoP

Supported Solid providers:
- Inrupt PodSpaces (https://storage.inrupt.com/)

The extension implements full Solid protocol compatibility including:
- OAuth 2.0 with PKCE authentication flow
- DPoP (Demonstration of Proof-of-Possession) tokens
- ECDSA P-256 key generation and JWT signing
- Resource access control and permissions
